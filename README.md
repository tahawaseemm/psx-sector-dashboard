# PSX Sector Performance Dashboard

## Business Question
How has KSE-100 sector leadership rotated month-to-month in 2026, 
and which sectors/stocks are driving the current rally?

## Data Source
Compiled from publicly reported PSX market figures (Express Tribune, 
Eurasia Review, and PSX Data Portal summaries), Feb–Jun 2026.

## What I Built
An interactive dashboard tracking:
- KSE-100 index trend over 5 months
- Month-by-month sector leaders vs. laggards
- Top index-heavy contributing stocks
- Key takeaways on market drivers (liquidity, geopolitics, rate outlook)

## Tools Used
HTML/CSS/JS, Chart.js

## Key Finding
Sector leadership rotated each month (cement/textile → banking/fertiliser 
→ broad-based auto/energy/banking), suggesting the FY26 rally was 
liquidity-driven and broad-based rather than concentrated in one sector.
